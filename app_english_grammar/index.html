영어 문법, 어디서부터 시작해야 할지 막막하셨나요?

섹션별 단계 학습부터 틀린 문제 반복까지, 체계적으로 완주를 돕는 Don't Know Grammar와 함께하세요.

단순 암기가 아닌, 문제를 풀고 틀리고 다시 맞히는 과정으로 진짜 문법 실력을 쌓습니다.

구독하면 모든 문법 섹션을 제한 없이 학습할 수 있어요.

How it Works

섹션 선택부터 통계까지, 3단계로 완주하는 영어 문법 학습

01

내게 맞는 문법 섹션 선택

수십 개의 문법 카테고리 중 오늘 공부할 섹션을 골라 맞춤형 퀴즈를 시작합니다.

02

퀴즈 풀기 → 정답 확인

4지선다 또는 입력형 문제를 풀고, 정답과 오답 해설을 즉시 확인해 틀린 이유를 파악합니다.

03

통계로 학습 진도 체크

날짜별 학습 기록과 정답률 통계로 내 약점 문법과 성장 과정을 한눈에 볼 수 있습니다.

Features

체계적으로 풀고 오래 기억하는 영어 문법 학습 경험

섹션별 체계적 학습

문법 카테고리를 섹션별로 나눠 순서대로, 또는 원하는 항목만 골라 집중 학습합니다.

학습 통계 & 달력 뷰

날짜별 학습 현황을 달력으로 시각화하고, 퀴즈 결과 목록으로 틀린 문제를 추적합니다.

즉각 피드백 시스템

정답·오답 팝업이 즉시 표시되어 문제를 풀면서 바로 개념을 교정하고 기억을 강화합니다.

합리적 구독 모델

무료로 시작해 핵심 기능을 체험한 뒤, 구독으로 모든 문법 섹션 무제한 학습을 이어갑니다.
