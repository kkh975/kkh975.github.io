How it Works

AI 카드 생성부터 간격 반복까지, 3단계로 완주하는 암기 학습

01

카테고리 생성 & AI 카드 자동 생성

학습 카테고리를 만들고 AI로 카드를 자동 생성하거나 직접 추가합니다. AI 번역 보조로 답안 입력도 빠르게 완료됩니다.

02

간격 반복 플래시카드 학습

카드를 보고 난이도(Again/Hard/Good/Easy 또는 Fail/Pass)를 평가하면, 기억 곡선에 맞는 복습 간격이 자동으로 조정됩니다.

03

꾸준한 복습으로 장기 기억화

설정한 복습 간격에 따라 적절한 시점에 카드가 다시 등장해 기억을 강화하고 장기 보존을 돕습니다.

Features

AI와 과학적 반복으로 효율적으로 암기하는 학습 경험

AI 카드 자동 생성 & 번역 보조

키워드만 입력하면 AI가 카드를 자동으로 생성하고, 뜻도 자동 추천해 카드 제작 시간을 획기적으로 줄여줍니다.

과학적 간격 반복 학습

Normal(4단계)·Easy(2단계) 평가 모드로 기억 상태에 맞게 복습 간격을 자동 조절해 효율적인 장기 기억을 만듭니다.

유연한 카드 관리

카테고리별 카드 목록 검색, 스와이프 삭제, 보관 기능으로 방대한 암기 카드를 체계적으로 관리합니다.

합리적 구독 모델

무료로 시작해 일일 제한 내에서 광고와 함께 체험하고, 구독으로 무제한 카드와 모든 기능을 사용합니다.
