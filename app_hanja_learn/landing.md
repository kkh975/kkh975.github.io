# Hanjabelle 랜딩 페이지

## 2,136자 상용한자, 아름답게 완주하기

일본어 한자 학습, 어디서부터 시작할지 막막하셨나요?
자동 복습부터 실전 테스트까지, 단계별로 완주를 돕는 Hanjabelle과 함께하세요.

단순 암기가 아닌, 쓰기·스토리·연관 단어로 입체적으로 익혀 오래 기억되는 학습 경험을 제공합니다.
설치 후 2주간은 로그인 없이 모든 기능을 자유롭게 체험할 수 있어요.

## How it Works

1. 내게 맞는 오늘의 리스트 선택
   새로 배울 한자를 확인하고, 이미 아는 글자는 [이미 알고 있음]으로 바로 통과해 맞춤형 학습 리스트를 확정합니다.

2. 플래시카드 → 실전 테스트
   앞면(한자)·뒷면(의미/이미지) 카드로 익힌 뒤, 음독/훈독 ↔ 한자 4지 선다 퀴즈로 바로 실전 감각을 체크합니다.

3. 수집 현황과 기록으로 동기 부여
   학습 완료한 한자는 도감(Grid)에서 즉시 활성화되고, 날짜별 타임라인에 학습 로그가 쌓여 진행 상황을 한눈에 볼 수 있습니다.

## Features

1. 🎯 단계별 학습 플로우
   자동 복습 → 큐레이션 → 플래시카드 → 테스트 → 완료까지 이어지는 구조로 누락 없이 반복 학습합니다.

2. 📚 도감 수집 시스템
   2,136자 상용한자 현황을 컬러로 표시해 달성도를 시각화하고, 활성화된 한자만 선명하게 확인합니다.

3. ✍️ 개별 한자 심화
   따라쓰기, 스토리, 연관 단어, 시험 문제로 한 글자씩 깊이 있게 익히며, 탭 어디서든 상세 화면에 바로 접근합니다.

4. 🔒 합리적 접근 제어
   설치 후 2주 체험(로그인 불필요) → 로그인 후 2주 자유 이용 → 이후 Basic은 일일 학습 2회·광고 시청, Premium은 무제한 이용으로 지속 학습을 지원합니다.
