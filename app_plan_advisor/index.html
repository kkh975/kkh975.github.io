자연어로 입력하면 AI가 자동으로 분류하고, 달력 뷰로 한눈에 파악하게 해주는 도라도라와 함께하세요.

반복 일정과 알림까지 한 번 설정하면 매번 신경 쓸 필요 없습니다.

프리미엄으로 AI 자연어 입력을 활성화하면 메모처럼 적어도 일정이 완성됩니다.

How it Works

입력부터 달력 파악, 알림까지 3단계로 완전 정리

01

일정·할일 빠르게 추가

직접 입력하거나, 프리미엄에서 자연어로 적으면 AI가 자동으로 분류해 날짜·시간까지 파악해 즉시 등록합니다.

02

달력으로 전체 일정 파악

오늘의 할일과 일정을 달력 뷰에서 한눈에 확인하고, 반복 일정도 자동으로 표시됩니다.

03

알림으로 놓치지 않기

30일 이내 알람이 자동으로 관리되고, 앱 실행 시 재동기화되어 항상 정확한 알림을 받습니다.

Features

AI와 달력으로 일정을 완전히 파악하는 관리 경험

달력 기반 일정 & 할일 통합

일정과 할일을 하나의 달력 뷰에서 통합 조회하고, 반복 일정은 RRULE 기반으로 정확하게 계산합니다.

AI 자연어 입력 (프리미엄)

자연어로 입력하면 AI가 일정인지 할일인지 자동 분류하고 날짜·시간까지 파악해 즉시 등록합니다.

스마트 알림 관리

Rolling Window 방식으로 30일 이내 알람만 OS에 등록해 배터리를 아끼면서도 정확한 알림을 보장합니다.

합리적 구독 모델

무료로 기본 일정·할일 관리를 시작하고, 프리미엄 구독으로 AI 자연어 입력을 무제한 이용합니다.
