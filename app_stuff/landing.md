# Repeta 랜딩 페이지

## 한 번 외운 지식, 평생의 자산으로.

눈으로만 훑고, 다 외웠다고 착각하셨나요?
이제는 과학적인 방법으로, 진짜 ‘내 것’으로 만들 시간입니다.

‘나중에 봐야지’ 쌓아둔 지식이 너무 많다면.
시간은 없는데, 무엇을 먼저 복습해야 할지 막막했다면.

Repeta는 당신의 두뇌가 잊기 직전에 다시 알려주는 가장 지능적인 암기 시스템입니다.

## How it Works

1. 기억할 내용 등록
   암기하고 싶은 지식(단어, 개념, 공식 등)을 나만의 플래쉬 카드로 간편하게 만듭니다.

2. 최적의 타이밍에 복습
   Repeta의 알고리즘이 당신의 망각 주기를 계산하여, 가장 효과적인 시간에 복습 알림을 보내줍니다.

3. 테스트와 기억 강화
   알림에 맞춰 카드를 확인하고, 아는 것과 모르는 것을 구분하며 지식을 장기기억으로 전환합니다.

## Features

1. 🧠 뇌과학 기반의 자동 복습 스케줄
   가장 효율적인 시점에 복습할 내용을 알아서 알려주어, 최소한의 노력으로 최대의 암기 효과를 만듭니다.

2. ✨ 쉽고 빠른 나만의 암기 카드
   텍스트와 이미지를 활용해, 어떤 지식이든 단 몇 분 만에 나만의 학습 카드로 만들 수 있습니다.

3. 💻 언제 어디서든 이어하는 학습
   이동 중에는 스마트폰으로, 책상에서는 PC로. 모든 기기에서 학습 내용이 실시간으로 동기화됩니다.

4. 📊 한눈에 보는 나의 성장 리포트
   내가 얼마나 학습했는지, 어떤 카드를 어려워하는지 직관적으로 파악하고 학습 동기를 유지할 수 있습니다.
