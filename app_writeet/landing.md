# Writeet 랜딩 페이지

## 매일 쓰는 외국어, 진짜 실력이 되다.

외국어 공부, 눈으로만 하셨나요?
이제는 직접 쓰면서 유창함을 완성할 시간입니다.

문법책만으로는 부족했다면.
학원은 부담스럽고, 매일 꾸준한 체계적인 연습이 필요했다면.

Writeet은 당신의 수준에 맞는 문법 문제를 통해, 매일 정확하게 쓰는 법을 훈련하는 가장 체계적인 방법입니다.

## How it Works

1. 내게 맞는 레벨 선택
   왕초보부터 고급까지, 현재 나의 실력에 맞는 문법 레벨을 선택하여 학습을 시작합니다.

2. 매일 새로운 문제 풀이
   매일 제공되는 오늘의 문법 문제를 풀고 직접 문장을 완성하여 제출합니다.

3. 즉시 채점 및 기록 확인
   제출 즉시 채점 결과를 확인하고, 나의 오답과 성장 과정을 한눈에 기록으로 관리합니다.

## Features

1. 🎯 체계적인 레벨별 문법 학습
   기초부터 고급까지, 나에게 꼭 맞는 난이도로 시작하여 문법의 빈틈을 꼼꼼하게 채웁니다.

2. ✍️ 매일의 문장 완성 챌린지
   하루 한 문제씩, 부담 없는 훈련을 통해 문법 규칙을 자연스럽게 체득하고 응용력을 키웁니다.

3. 🚀 즉각적인 피드백과 채점 시스템
   내가 만든 문장이 맞았는지 바로 확인하고, 정확한 표현을 익히며 실력을 빠르게 향상시킵니다.

4. 📈 한눈에 보는 나의 성장 리포트
   어떤 문법에 강하고 약한지 파악하고, 성취감을 느끼며 학습 동기를 유지할 수 있습니다.
